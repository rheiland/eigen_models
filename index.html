PhysiCell eigen-models

This site provides an introduction to PhysiCell using eigen-models. More of a reference model than an eigen-model, let's begin with a growing monolayer.

  • Repulsion: cell-cell repulsion parameter
  • Max Cells: max # of cells in the monolayer
  • Win Size: Window size (dynamically editable to zoom in)
  • Split type: how daughter cells are positioned
  • Seed: PRNG seed
  • (Cells) Color Mode: white boundaries or colored by gamma
Each cell has a "gamma" scalar value, [0,1], which is an approximation to its free surface boundary, e.g., 0=no free boundary; neighbor cells totally overlap it; 1=100% free
