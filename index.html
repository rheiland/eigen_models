PhysiCell eigen-models

This site provides an introduction to PhysiCell using eigen-models.

More of a reference model than an eigen-model, let's begin with a growing monolayer.

  • Repulsion: cell-cell repulsion parameter
  • Max Cells: max # of cells in the monolayer
  • Win Size: Window size (dynamically editable)
  • Daughters: how daughter cells are positioned
  • Seed: PRNG seed
  • dt_mechanics: time step for mechanics
  • Grow Rate R: actual cell growth rate, based on 11-cell relaxation (0.011325 is actual rate, not the 5x slower)
  • Modulo Frame: only save/plot every modulo computed frame (e.g., 5th, 10th, 20th)
  • Color: white boundaries, or gamma, or ID

Each cell has a "gamma" scalar value, [0,1], which is an approximation to its free surface boundary, e.g., 0=no free boundary; neighbor cells totally overlap it; 1=100% free
